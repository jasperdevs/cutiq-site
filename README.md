Hey 👋 As we're in a private repo, no need to be overly formal. First of all, thank you for purchasing AppView and supporting my work!

Just to reassure you, this is not a project that was thrown together in a weekend. It came from my personal need when I was rebuilding a website for my app Ripples. I wasted so much time overthinking every little detail, and I knew that soon I would need to do it all over for my second app. So I've decided to pack everything into a reusable and customizable template.

## Getting Started

Head over to <a href="https://appview.dev/docs">the documentation</a>. It will walk you through everything from running the project locally to customizing everything and eventually deploying it to production.
